## Business description (short)

```
AMRShield sells AI-powered antibiotic stewardship as a professional service. 
Gemini agents on Cloud Run handle clinical recommendations and self-audit safety 
checks for each patient case. We generate revenue through 14-day paid hospital 
and pharmacy college pilots. AI executes tool calls, audits, and dashboard 
updates; humans handle sales, onboarding, and physician governance.
```

---

## AI-native operations (how AI runs the business)

```
• Clinical Agent processes 100% of customer case submissions autonomously
• Self-Audit Agent runs Phoenix MCP safety pipeline on every recommendation
• No manual per-case coding — agents call CrCl, AWaRe, interaction, antibiogram tools
• Audit Console logs PASS/FLAG/HOLD without human data entry
• Cloud Run auto-scales production workload
• Founder focuses on sales/support; agents execute operational decisions
```
